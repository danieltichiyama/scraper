Release notes for Optimizely updates

This topic lists Optimizely updates, delivered as NuGet packages and services. You decide which updates apply to your project; see Installing Optimizely updates.

Select a product, package, or service in the left menu, and then select one of the following filters from Item type and click Filter.

  • Bug. Display bug fixes.
  • Critical bug. Display only critical bug fixes.
  • Feature. Display only new features (all features).
  • UI Feature. Display only end-user (user interface) features.

Note: NuGet packages listed here may not be immediately available in the Optimizely NuGet feed.

CMS-20848
  Page tree performance issue

Page Tree was very slow when there were more than 1000 child pages under one parent page.

Performance issues occurred after upgrading Chrome from version 94 to version 95. Firefox was unaffected.

EPiServer.CMS.UI 11.36.4;

CMS-20604
  Content selector does not show information about expired pages

When you expired a page, the content selector did not display information about the expired page. An editor should know that the page is expired with icons in the page navigation tree before selecting content.

EPiServer.CMS.UI 11.36.4;
Nov 30, 2021
CMS-18990
  Target language was latest published version instead of master language

When translating, the language that had the latest published version was used to translate from instead of using the master language.

EPiServer.CMS.UI 11.36.4;
Nov 30, 2021
CMS-15759
  Unable to create new page after setting Inherit settings from the parent page option

When you enabled Inherit settings from the parent page in Available Languages, all available languages were lost on the current page. This caused the inability to create subpages.
